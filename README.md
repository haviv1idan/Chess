# Prerequisites:
- python 3.11.4
- pip

# Installation:
1. clone this repository:
  ```bash
    git clone https://github.com/haviv1idan/Chess
  ```
2. Navigate to the project directory:
  ```bash
    cd Chess  
  ```
3. Install Dependencies:
  ```bash
    pip install -r requirements.txt
  ```

# Usage:
  ```Bash
    python main.py 
  ```
